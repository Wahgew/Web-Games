# Rock Paper Scissors Website

## Description
This is a simple website that allows you to play the game "Rock Paper Scissors" in your web browser. 
The website provides buttons to select your choice (rock, paper, or scissors), and it displays the player's choice, the computer's choice, and the winner of the game.

## Installation
1. Clone or download the repository to your local machine.
2. Ensure that all the files, including the images, are saved in the same folder/directory.

## Usage
1. Open your web browser.
2. In the address bar, type the full file path to the `RPSWebsite.html` file. For example:

Replace "D:/University/My%20Personal%20Programs/WebGames/src/RPSWebsite.html" with the actual path to the folder where you saved the files.
3. Press Enter to load the website in your browser.
4. Play the game by clicking on the "ROCK," "PAPER," or "SCISSORS" buttons.
5. The website will display the player's choice, the computer's choice, and the winner of the game.

## Compatibility
This website should be compatible with most modern web browsers.
