# Black Jack Website

## Description
This is a simple website that allows you to play the game "Black Jack" in your web browser. 
The website provides buttons to select your choice (Hit or Stay), and it displays the player's and computers hand after the winner is decided.
For more information on how to play Black Jack Vist https://www.cs.mcgill.ca/~rwest/wikispeedia/wpcd/wp/b/Blackjack.htm

## Installation
1. Clone or download the repository to your local machine.
2. Ensure that all the files, including the images, are saved in the same folder/directory.

## Usage
1. Open your web browser.
2. In the address bar, type the full file path to the `BlackJack.html` file. For example:

Replace "D:/University/My%20Personal%20Programs/WebGames/src/Black%20Jack/BlackJack.html" with YOUR actual path to the folder where you saved the files.
3. Press Enter to load the website in your browser.
4. Play the game by on hit or stay. Aim to get 21 (Black Jack) to win.
if you go over it's a bust and you lose. And if you amount is less than the computer you lose.
5. The website will display the player's cards and Computers cards. After the player hit stays it will display
the card sum of both player and computer aswell as displaying the winner.

## Compatibility
This website should be compatible with most modern web browsers.
